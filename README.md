# SQL - Prática de Consultas e Relatórios de Estoque

Projeto de prática em SQL utilizando SQLite.

 Objetivo

Simular consultas utilizadas em relatórios de estoque e análise básica de dados,
com foco em:

- Filtros com WHERE
- Operadores lógicos (AND / OR)
- LIKE
- BETWEEN
- IN
- Funções agregadas (COUNT, SUM, AVG, MIN, MAX)
- Cálculos financeiros (preço × estoque)

 Estrutura

- 01_schema.sql → Criação da tabela
- 02_dados.sql → Inserção de dados
- 03_consultas_filtros.sql → Filtros simples e compostos
- 04_consultas_like_between_in.sql → LIKE, BETWEEN e IN
- 05_funcoes_agregadas.sql → Relatórios com funções matemáticas

 Tecnologias

- SQLite
- SQL padrão

---

Como executar o projeto

1. Criar o banco utilizando o arquivo schema.sql
2. Inserir os dados com dados.sql
3. Executar as consultas nos arquivos consultas_*.sql

Projeto desenvolvido para prática e evolução na área de Suporte SQL / ERP.
